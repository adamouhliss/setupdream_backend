Database Setup Instructions for Setup dreams
=============================================

Your PostgreSQL database configuration:
- Host: localhost
- Port: 5432
- User: postgres
- Password: ChatBot123!
- Database: setupdream

To set up the database connection:

1. Create a .env file in the backend directory with:
   DATABASE_URL=postgresql://postgres:ChatBot123!@localhost:5432/setupdream

2. Or set the environment variable directly:
   - Windows PowerShell: $env:DATABASE_URL="postgresql://postgres:ChatBot123!@localhost:5432/setupdream"
   - Windows CMD: set DATABASE_URL=postgresql://postgres:ChatBot123!@localhost:5432/setupdream
   - macOS/Linux: export DATABASE_URL="postgresql://postgres:ChatBot123!@localhost:5432/setupdream"

3. Run the migration script:
   python migrate_to_postgres.py

4. Start your FastAPI server:
   python -m uvicorn app.main:app --reload
